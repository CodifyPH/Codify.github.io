CSS, or Cascading Style Sheets

CSS, or Cascading Style Sheets, is like the "fashion" for web pages. It controls how things look, like colors, fonts, and spacing. You use CSS to make your website stylish and organized. It works with HTML, the language that structures your content. With CSS, you decide how your webpage should appear on different devices and even add cool animations. It's what makes your website visually appealing and user-friendly.

Source code for macbook and windows

  • For windows: ctrl + shift + C
  • For Mac OS: Cmd + Opt + I

To connect your CSS to your HTML, follow these simple steps:

  1. Create a CSS File:

    Open your text editor (like Notepad, VS Code, or Sublime Text) and create a new file. Save it with a ".css" extension. For example, name it styles.css. This is where your CSS code will go.
    styles.css:

  2. Link CSS File to HTML:

    Open your HTML file (for example, index.html) in the same text editor. Inside the (head) section, add a (link) tag to connect your HTML file with the CSS file. Adjust the href attribute to point to your CSS file.
    index.html:

  3. Verify Connection:

    Save both files and open the HTML file in a web browser. You should see your HTML content styled according to the rules specified in your CSS file.

CSS spacing

Achieving spacing in CSS is more convenient and efficient compared to the somewhat challenging and time-consuming task of editing (br) code. In this code, the focus is on the margin property for creating space around an element and the padding property for creating space within an element. These styles can be applied to HTML elements as needed for your spacing requirements.

These examples should provide a clear illustration of how each type of selector works. Remember that in practical use, you would apply these styles to corresponding HTML elements with the specified tags, classes, or IDs

  1. Type (or tag) Selector: Selects all paragraphs and gives them a red color.
  2. Class Selector: Selects all elements with the class "highlight" and gives them a yellow background.
  3. ID Selector:Selects the element with the ID "main-heading" and makes its text bold.

Inline CSS and Custom CSS

  1. Inline CSS:
    • Definition: Inline CSS is when you apply styles directly to an HTML element using the style attribute.
    • Example:
    • Explanation: Styles are specified within the HTML tag itself. While it's convenient for small changes, it can make the HTML less clean and harder to maintain, especially for larger
  2. Custom CSS:
    • Definition: Custom CSS involves placing all your styles in a separate CSS file. You link this file to your HTML, keeping the styling separate from the HTML structure.
    • Example:
    • Explanation: Styles are kept in a separate file (commonly with a .css extension), making the HTML cleaner and the styling reusable across multiple pages. It promotes better organization and maintenance of your website's code.

    In summary, inline CSS involves styling directly within HTML tags using the style attribute, while custom CSS involves keeping all your styles in a separate file and linking it to your HTML document. Custom CSS is generally preferred for larger projects as it promotes cleaner, more maintainable code.
